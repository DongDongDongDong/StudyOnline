This app can provide the learning function,Such as online learning video, offline download, etc

#### 功能介绍
一款在线学习小软件
1）标签会根据文字大小自动伸缩   
1）可以选择第三方进行登录，如微信，QQ，新浪微博等   
2）选择感兴趣的课程，点击可进行视频播放。    
3）可以离线下载或收藏课程。   
4）升级版本，清除缓存，设置网络环境等。
